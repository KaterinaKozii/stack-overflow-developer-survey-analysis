**Stack Overflow Developer Survey 2025 Analysis**

**Опис проєкту**

Метою цього проєкту є аналіз великого датасету Stack Overflow Developer Survey 2025 для дослідження сучасного стану IT-ринку, професійних тенденцій, рівня компенсацій, популярності технологій та особливостей кар'єрного розвитку спеціалістів.
У межах аналізу було досліджено відповіді понад 49 тисяч респондентів з різних країн світу з використанням бібліотеки Pandas та методів дослідницького аналізу даних (EDA).

**Бізнес-завдання**

Дослідження було спрямоване на отримання відповідей на наступні питання:
- Які технології та мови програмування є найбільш популярними серед розробників.
- Як відрізняється рівень компенсації залежно від професії, освіти та досвіду.
- Які фактори впливають на професійний розвиток спеціалістів.
- Які тенденції спостерігаються на міжнародному IT-ринку.
- Як змінюються підходи до віддаленої роботи та кар'єрного зростання.

**Етапи аналізу**
1. Завантаження та первинне дослідження великого датасету.
2. Перевірка структури даних та оцінка їх якості.
3. Аналіз пропущених значень.
4. Очищення та підготовка даних до аналізу.
5. Фільтрація респондентів за різними критеріями.
6. Групування даних та розрахунок агрегованих показників.
7. Аналіз компенсацій, освіти, досвіду та популярності технологій.
8. Формування аналітичних висновків та інтерпретація результатів.

**Основні результати**
- Проаналізовано міжнародне опитування розробників із вибіркою понад 49 тисяч респондентів.
- Виявлено залежності між рівнем компенсації, професійним досвідом та освітою.
- Досліджено популярність мов програмування та технологічних стеків.
- Проаналізовано особливості віддаленої роботи та кар'єрних траєкторій спеціалістів.
- Отримано практичні висновки щодо тенденцій сучасного IT-ринку.

**Бізнес-рекомендації**
- Використовувати результати дослідження для аналізу ринку праці та конкурентного середовища.
- Враховувати тенденції популярності технологій при плануванні професійного розвитку.
- Використовувати дані щодо компенсацій для оцінки ринкових зарплатних очікувань.
- Регулярно проводити аналіз галузевих досліджень для моніторингу змін на ринку.
